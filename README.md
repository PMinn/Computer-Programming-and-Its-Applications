# 大學周遭交通事故分析

## 依賴套件
```
matplotlib
numpy
pandas
selenium
flask
openpyxl
```

## chromedriver
下載[chromedriver](https://googlechromelabs.github.io/chrome-for-testing/#stable)並將檔案放入以下路徑

```
/chromedriver/chromedriver.exe
/chromedriver/LICENSE.chromedriver
```

## 執行
```
cd src
python server.py
```
